# Hashtable Benchmarks

Benchmarks for comparing hashtable implementations.

1. Build:

```shell
bazel build -c opt :hashtable_benchmarks
```

2. Run:

```shell
./bazel-bin/hashtable_benchmarks --benchmark_format=json > benchmark-results.json
```
